# Demo Video

This folder contains a scripted Remotion demo for the public project. It uses simulated Slack data and intentionally avoids real workspaces, tokens, user IDs, and message content.

Preview:

```sh
npm run demo:preview
```

Render:

```sh
npm run demo:render
```

The rendered video is written to:

```text
demo/slack-api-cli-demo.mp4
```
